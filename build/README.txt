# qprop.c

This folder contains precompiled binaries of qprop.c for Windows, Linux and MacOS ARM,
along with convenient Python and Julia wrappers to help you get started quickly.

To verify that everything is working correctly, try running one of the provided examples.
Open a terminal and execute one of the following commands:
    python3 example_python.py
    julia example_julia.jl
If successful, the output should display the following results:
    Thrust: 3.22175 N
    Torque: 0.02969 N-m"
